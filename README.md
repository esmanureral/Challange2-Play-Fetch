3) **_Sabit hızla ilerletmek(toplar)_**
4) **_Prefabs yapmak ve çarpışılınca yok edilmesini sağlamak_**
5) **_Tuşa basılınca köpek yollamak_**
6) **_Top yere değerse "Game Over!" yazdırmak_**
